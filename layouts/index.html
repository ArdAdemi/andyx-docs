About Andy X

Andy X is an open-source distributed streaming platform designed to deliver the best performance possible for high-performance data pipelines...

Get Started

To run Andy X locally, the fastest way is to do it using Docker. You can download Andy X images from Docker Hub, and run it using Docker commands.

Install, Deploy and Upgrade

Built from the beginning to be upgradable, and deployable anywhere and anytime, in the cloud or on-prem.

Andy X for Developers

The best data streaming engine for developers, enabling new architecture style possibilities like Reactive Architecture, by utilizing Andy X libraries and Streams.

How To

By clicking Get Started, you can learn more about Andy X, and also checking the examples and learn how to run, deploy and making it ready for your project.
